## Cara Menjalankan Project

1. Clone repository ini.
2. Install library: `pip install -r requirements.txt`.
3. **PENTING:** Download Model Machine Learning.
   Karena ukuran file model terlalu besar untuk GitHub, silakan download file `.pkl` melalui link berikut:
   https://drive.google.com/drive/folders/1KTOO5I_vyJJGg6evXwhBhRhRByEayFBr?usp=sharing 
4. Setelah didownload, letakkan file tersebut di folder:
   - `backend/tickets/utils/`
5. Jalankan server: `python manage.py runserver`.
